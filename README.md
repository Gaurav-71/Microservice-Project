# Microservice-Project

7th Sem Microservice Spring Boot Project

## Installation Steps

- Prerequisites

  - [MySQL](https://dev.mysql.com/downloads/mysql/)

  - [phpMyAdmin]()

  - `brew install maven`

- Springboot

  `brew tap spring-io/tap`

  `brew install spring-boot`

## Random Steps

- Run apache server : `sudo apachectl start`
